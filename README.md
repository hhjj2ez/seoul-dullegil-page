# 서울둘레길 모바일 안내 페이지

GitHub Pages 배포용 정적 웹페이지입니다.

## 파일 구성

- `index.html`: 모바일 안내 페이지 본문과 스타일
- `script.js`: 코스 난이도 필터와 하단 내비게이션 활성화 스크립트
- `assets/seoul-dullegil-hero.png`: 히어로 이미지
- `.nojekyll`: GitHub Pages에서 정적 파일을 그대로 제공하도록 하는 설정 파일

## GitHub Pages 설정

1. 이 폴더를 GitHub 저장소에 push합니다.
2. GitHub 저장소의 `Settings > Pages`로 이동합니다.
3. `Build and deployment`에서 `Deploy from a branch`를 선택합니다.
4. Branch는 `main`, Folder는 `/ (root)`로 설정합니다.
5. 저장 후 표시되는 Pages URL로 접속합니다.
